## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/keshaba-dev/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Open `index.html` in a web browser.

## Deployment Guide
To deploy this portfolio to a web server, follow these steps:
1. Copy the project files to your server's document root.
2. Ensure the server is configured to serve static files.
3. Access your portfolio via the server's URL.
